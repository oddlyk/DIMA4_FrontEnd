목록과 관련된 태그

  - nbsp 없이 자동 들여쓰기가 됨
ul: 순서가 없는 리스트
  - li태그를 통해 목록에 대한 항목 기술(●)
  - type=
    - "square": 사각형 ... ol: 순서가 있는 리스트
  - li태그를 통해 목록에 대한 항목 기술(숫자)
  - ol내의 type 속성을 통해 항목 모양 변경 가능 (default:숫자(1))
    - "1":숫자, "a":영문 소문자, "A": 영문 대문자, "i": 로마숫자 소문자, "I": 로마숫자 대문자   - ol내의 start 속성을 통해 항목의 시작 값 변경 가능 (default:1)
  - ol내의 reverse 속성을 통해 항목 역순 출력 가능 dl: 정의에 관련됨
  - dt태그(제목)과 dd태그(설명)를 통해 정의 기술
  - 숫자나 문자없는 항목 생성

실습

ul>li

  • 항목1
  • 항목2
  • 항목3
  • 항목4

ol>li

  1. 항목1
  2. 항목2
  3. 항목3
  4. 항목4

  • 항목4
