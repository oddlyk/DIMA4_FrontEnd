글자와 관련된 요소(Elemnet)

p태그 = 기본적인 데이터(글씨)를 표현
br태그 = 일반적으로 내용을 다음줄로 옮김(singleton tag)
hr = 수평선 태그

hn태그 = 제목을 표기(n=1~6)
  - h1 : 로고, 표지 제목 등
  - h2 : 대제목, 단락제목 등
  - h3 : 중제목, 페이지 제목 등
  - h4 : 소제목, 문단 제목
  - h5 : 문단제목
  - h6 : 주석제목

강조
  - strong : 굵게
  - b : 굵게
  - i : 기울어짐
  - em : 이탤릭체
  - u : 밑줄
  - s : 취소선

mark 태그 : 형광펜 span 태그 : 줄바꿈 없이 일부 영역 묶기
div 태그 : 단락을 통째로 묶음 (텍스트 영역 블록처리)

ruby내용rt 주석

일반문구에sub 아래첨자

일반문구에sup 위첨자

블록태그(block) : 주로 다른 콘텐츠를 포함함

h1 제목

h2 제목

h3 제목

h4 제목

h5 제목
h6 제목
