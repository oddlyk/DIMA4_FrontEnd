폰트

  • font-family: 글꼴1 글꼴2
    • 글꼴지정
    • 대중적인 글꼴을 사용하는 것을 권장
    • 글꼴1이 제대로 적용되지 않았을 때 글꼴2가 적용
  • font-size: 값
    • 글꼴 크기 지정
    • 절대 크기: 브라우저에서 지정
    • 상대크기: 부모요소 글자크기를 기준으로... larger|smaller
    • 크기: em(해당 글꼴의 대문자 M 너비를 기준으로 %), ex(해당 글꼴의 소문자 x 높이를 기준으로 %), px, pt
    • 백분율: 부모 요소의 글자 크기를 기준으로 % 계산
  • font-weight: (100~900)
    • 굵기지정
    • 기본값 : 400
    • 100~900 사이의 값, bold, lighter, bolder 등
  • font-varient: normal|small-cap
    • 소문자 크기로 대문자를 표현함
  • font-style: normal|italic|oblique
    • italic: 처음부터 디자인된 기울임꼴 (주로)
    • oblique: 표시만 기울어진 형태
  • **글꼴 속성을 한꺼번에 묶어 작성할 수도 있음**
    • font: style variant weight size family ... 순서로...
  • 이미지를 웹에서 class 요소로 가져오기(script를 이용)
    • 아래 내용이 header에 포함되어 있어야 class를 가진 요소를 이미지로서 웹에서 가져올 수 있음
    • script> src="https://kit.fontawesome.com/def97ee41b.js" crossorigin="anonymous"/script>


텍스트

  • color:: 글자색상
  • text-decoration: 밑줄 등을 넣거나 뺄때
  • text-transform: 대소문자 변경
  • text-shadow: 그림자 지정
  • white-space: 공백에 대한 처리
  • letter-spacing / word-spacing: 공백처리 (문자사이의 공백/단어사이의 공백)
  • direction: 글자가 써지는 방향(ltr(left to right) | rtl(right to left)
  • text-align: 수평정렬
  • text-justify: 정렬 시 남는 공백 처리
  • line-height: 줄간격
  • text-overflow: overflow 속성과 함께 사용. 지정 영역 내 글자가 흘러 넘칠경우의 표현 방법
    • text-overflow=clip
    • text-overflow=ellipsis
    • overflow 속성
      • overflow=auto: 흘러넘친 글자가 흘러넘친 채로 보여짐(default) = visible
      • overfow=hidden: 흘러넘치면 숨겨짐
      • overfow=scroll: 스크롤바 생성
