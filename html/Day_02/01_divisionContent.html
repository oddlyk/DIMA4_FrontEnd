영역분할 태그

브라우저 내에 공통된 역할을 담당하는 요소들을 집합시켜 다른 요소와 구별하고,
그룹화된 요소를 브라우저의 특정 위치에 배치 시키기 위한 태그
  - div, span
  - header, footer, article, section 등     - 위 4개는 div의 진화형

div 태그 : 영역분할에 사용되는 대표적인 태그

블럭요소의 특징
  • 위에서 아래로 내용이 쌓이는 구조
  • 경계가 있음
  • 너비와 높이가 적용됨
    • css를 이용해 너비와 높이, 위치 조정 가능
  • div 태그 외에도 sementic 태그인 header, footer, section, article, hn, p, ol, li, dl ... 등도 블럭요소임

span 태그 : 인라인 요소이면서 인라인 요소들의 영역분할 담당

인라인 요소의 특징
  • 왼쪽에서 오른쪽으로 콘텐츠가 쌓이는 구조
  • 경계 속성이 없음
    • 경계를 만들고자 하면 css 속성을 사용해야 함
  • 인라인 요소는 너비와 높이를 기본값으로 줄수 없음
    • css 속성을 사용해야 함
  • b, i, u, img, span, (텍스트) 등이 인라인 요소
