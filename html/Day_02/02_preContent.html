pre 요소

일반적인 요소들은 탭, 스페이스, 엔터가 아무리 요소 내에 많아도 하나로 처리한다.
프로그램등의 코드를 화면에 나타내야 할 때에는 pre 태그가 효과적이다.


pre가 없는 코드 (p코드 사용)

강제 띄어쓰기 기능을 활용한 들여쓰기 (추천하지 않음)
public class Test {
    public static void main(String[] args){
        System.out.println("Hello, world!");
    }
}


pre가 적용된 코드

        public class Test {
            public static void main(String[] args){
                System.out.println("Hello, world!");
        }
        }
