폼요소

사용자의 입력을 받아, 그 내용을 서버로 전송하기 위해 사용되는 태그
폼요소는 대부분 인라인 요소임
회원가입, 로그인, 비밀번호 변경, 게시글 입력, 댓글 입력 등에서 사용

input, textarea, select, label 등이 있음.
name 속성: 서버측 변수명의 역할

for 태그의 속성 중 

            1) method="get" (default)
                - 주소표시줄에 노출됨(개인정보 유출에 주의)
                - 다른 화면을 달라는 요청으로 사용 (다음 페이지로의 전환 등)
                - 많은 양의 데이터 전송 시에는 사용하지 않음
            2) method="post"
                - 주소표시줄에 노출되지 않음
                - 처리해 달라는 요청
                - 많은 양의 데이터 전송, 파일 첨부시 사용
                +) 제대로 전송신호를 보냈는가에 대해, 페이지 우클릭>검사>네트워크>페이로드 에서 전송여부, 전송 내용 확인 가능
